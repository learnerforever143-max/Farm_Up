## ğŸ“± Running the Mobile App

The FARM UP application is built with Flutter. To run it:

1. **Install Flutter**: Follow the official guide at https://flutter.dev/docs/get-started/install

2. **Get dependencies**:
   ```bash
   flutter pub get
   ```

3. **Run on emulator or device**:
   ```bash
   flutter run
   ```

4. **Build for release**:
   ```bash
   # Android
   flutter build apk
   
   # iOS (Mac only)
   flutter build ios
   
   # Web
   flutter build web
   ```
